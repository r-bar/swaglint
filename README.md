Yet another Swagger linter

## Install

Via NPM

``` bash
$ npm install swaglint
```

## Usage

``` bash
$ swaglint swagger.yml
$ swaglint partial.yml other.yml
$ cat cool-api.yml | swaglint --stdin
```
